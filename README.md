# [심화 실습 평가2] TCP 기반 원격 장치 제어 프로그램

본 프로젝트는 라즈베리파이4(Server)와 우분투 리눅스(Client) 간의 TCP/IP 통신을 활용하여 원격으로 하드웨어 장치를 제어하고 모니터링하는 시스템입니다.

---

## 1. 프로젝트 개요
* **서버(Server):** 라즈베리파이 4
    * 하드웨어 제어 및 데몬(Daemon) 프로세스 운영
* **클라이언트(Client):** 우분투 리눅스
    * 사용자 명령 전송 및 실시간 상태 모니터링
* **개발 환경:** C/C++, Linux (Ubuntu, Raspberry Pi OS)
* **빌드 시스템:** `make`

---

## 2. 주요 구현 내용
### 🛠 장치 제어 (Hardware Control)
* **LED:** On/Off 및 밝기 제어 (최대/중간/최저 3단계)
* **부저(Buzzer):** 클라이언트 명령을 통한 소리 On/Off
* **조도 센서:**  실시간 조도 값 확인
    * **자동 제어:** 빛이 감지되지 않으면 LED On, 감지되면 LED Off
* **7세그먼트:**  숫자(0~9) 표시 및 1초 단위 카운트다운
    * 0 도달 시 부저 울림 기능

### 💻 시스템 아키텍처
* **멀티 프로세스/스레드:** 여러 장치를 효율적으로 동시 제어
* **공유 라이브러리:** 각 장치 제어 기능을 `.so` 파일로 작성하여 업그레이드 및 확장성 확보 (Dynamic Library)
* **데몬 프로세스:** 서버를 백그라운드 서비스 형태로 구현
* **시그널 처리:** `SIGINT` 외의 신호에 대해 강제 종료 방지 및 안정적인 종료 처리
